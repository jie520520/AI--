### 1. 后置输入优先去重逻辑 ⭐⭐⭐

这是Tkinter版本的**核心特性**，现已完美移植！

#### 工作原理

```python
def deduplicate_tiers(active_tier, tier_data):
    """
    核心：后置输入优先去重逻辑
    从其他档位中删除与活跃档位重复的号码
    """
    active_numbers = set(tier_data[active_tier])
    
    for tier in tier_data:
        if tier != active_tier:
            # 从其他档位中删除重复号码
            tier_data[tier] = [n for n in tier_data[tier] if n not in active_numbers]
    
    return tier_data
```

#### 使用场景

**场景1：手动输入时自动去重**


### 2. 属性自动分类

系统自动识别号码属性，无需手动分类。

#### 分类规则

| 属性 | 规则 | 示例 |
|------|------|------|
| **大单** | 25-49 且 单数 | 25, 27, 29, 31, 33... |
| **大双** | 25-49 且 双数 | 26, 28, 30, 32, 34... |
| **小单** | 1-24 且 单数 | 1, 3, 5, 7, 9, 11... |
| **小双** | 1-24 且 双数 | 2, 4, 6, 8, 10, 12... |

#### 代码实现

```python
def get_number_attribute(num):
    """获取号码属性"""
    is_big = num >= 25
    is_odd = num % 2 == 1
    size = '大' if is_big else '小'
    parity = '单' if is_odd else '双'
    return f"{size}{parity}"

def auto_classify_numbers(numbers):
    """自动将号码按属性分类"""
    classified = {'大单': [], '大双': [], '小单': [], '小双': []}
    for num in numbers:
        attr = get_number_attribute(num)
        classified[attr].append(num)
    return classified
```


### 3. 一键移动到档位

点击按钮将整个属性组的号码移动到指定档位。

#### 按钮布局

```

│  25 27 29          26 28 30          01 03 05     02 04  │
```

#### 操作流程

```
步骤1：原始号码区输入
  → 1 2 3 4 25 26 27 28

步骤2：系统自动分类
  → 大单: 25 27
  → 大双: 26 28
  → 小单: 01 03
  → 小双: 02 04

步骤3：点击"小单"的"1档"按钮
  → 小单的所有号码移动到1档
  → 如果其他档位有重复，自动删除
  → 1档变为活跃档位

步骤4：点击"大单"的"2档"按钮
  → 大单的所有号码移动到2档
  → 执行去重逻辑
  → 2档变为活跃档位

结果：
1档：[01 03]  ← 小单
2档：[25 27]  ← 大单
3档：[]
4档：[]
```

#### 代码实现

```python
# 按钮点击处理

    tier_name = '1档(最高)'

    # 添加新号码

    # 执行去重
    st.session_state.tier_data = deduplicate_tiers(tier_name, st.session_state.tier_data)
    # 刷新界面
    st.rerun()
```

## 🎓 教育价值

### 学习后置优先去重算法

```python
def deduplicate_tiers(active_tier, tier_data):
    """
    算法思想：
    1. 获取活跃档位的所有号码（Set）
    2. 遍历其他档位
    3. 从其他档位中过滤掉重复号码
    4. 返回去重后的数据
    
    时间复杂度：O(n×m)
    n = 档位数量（4）
    m = 平均每档号码数
    
    空间复杂度：O(n)
    """
    active_numbers = set(tier_data[active_tier])
    
    for tier in tier_data:
        if tier != active_tier:
            tier_data[tier] = [n for n in tier_data[tier] if n not in active_numbers]
    
    return tier_data
```
