## 🐛 常见问题排查

### 问题1: ModuleNotFoundError

**错误**:
```
ModuleNotFoundError: No module named 'lottery_core'
```

**原因**: 文件不在同一目录

**解决**:
1. 确保5个.py文件在同一文件夹
2. 在该文件夹中运行streamlit命令
3. 使用`cd`命令进入正确目录

### 问题2: ImportError

**错误**:
```
ImportError: cannot import name 'SuperLearningEngine'
```

**原因**: super_learning_engine.py文件缺失或版本不匹配

**解决**:
1. 检查文件是否存在
2. 重新下载super_learning_engine.py
3. 确保文件大小约22KB

### 问题3: AttributeError

**错误**:
```
AttributeError: 'SuperLearningEngine' object has no attribute 'ultra_learn'
```

**原因**: super_learning_engine.py版本过旧

**解决**:
1. 使用最新版super_learning_engine.py
2. 检查文件中是否有`ultra_learn`方法

### 问题4: KeyError: 'volatility'

**原因**: lottery_core_enhanced.py使用了错误的键名

**解决**:
1. 使用修复版lottery_core_enhanced.py
2. 查看KeyError修复说明.md
