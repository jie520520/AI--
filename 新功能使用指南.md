## 📊 改进1: Transformer深度学习回测

### 使用方法

```
1. 运行系统
   streamlit run lottery_app_v4_complete.py

2. 上传数据文件

3. 进入"🔄 历史回测"标签页

4. 在"选择回测类型"下拉菜单中，现在可以看到：
   ├─ AI融合预测
   ├─ 🆕 Transformer深度学习  ← 新增选项
   ├─ 特码TOP1
   ├─ 特码TOP3
   ├─ 特码TOP5
   ├─ 特码TOP10
   ├─ 大小
   ├─ 单双
   └─ 波色

5. 选择"Transformer深度学习"

6. 设置回测参数：
   - 回测期数：100期（推荐）
   - 预测号码数：38个

7. 点击"▶️ 开始回测"

8. 查看结果：
   准确率: XX.X%
   命中期数: XX/100
   理论随机: 77.55%
   超越随机: +XX.XX%
```
