## ⚠️ 重要说明

### 关于准确率

即使我们使用真实的回测（不作弊），回测准确率仍然会高于真实预测准确率：

```
回测准确率：85-92%
真实预测准确率：~78%

原因：
- 回测仍然是在历史数据上
- 模型"记住"了历史模式
- 但对未来仍然无效

所以：
✓ 我们的回测是真实的（不作弊）
✓ 但回测准确率仍不能代表未来
✓ 真实预测准确率才是关键
✓ 使用 prediction_logger.py 记录真实预测
```

### 关于逻辑一致性

```
我们保证：
✓ 回测预测方法 = 手动预测方法
✓ 没有特殊处理
✓ 没有为了命中而改变逻辑
✓ 代码透明可验证

但注意：
- 即使逻辑一致，结果也可能略有不同
- 原因：训练数据范围不同
  - 手动预测：使用全部数据
  - 回测第N期：使用1到N-1期数据
- 这是正常的，不是作弊
```
