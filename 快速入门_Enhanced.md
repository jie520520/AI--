## 🆕 新功能快速体验

### 功能1: 下注助手 💰

```
步骤：
1. 上传数据 → 运行AI预测
2. 进入「下注助手」标签页
3. 设置总预算(如1000元)和赔率(如47.0)
4. 系统自动分配号码到4个档位
5. 点击「计算最优下注金额」
6. 点击「复制投注格式」

结果：
✓ 每个号码的精确下注金额
✓ 预期中奖利润
✓ 格式化投注文本
✓ 一键复制

示例输出：
澳特: 39 12 25 各下: 23.45
澳特: 18 07 31 各下: 18.20
澳特: 44 03 各下: 15.50
```

### 功能2: AI辅助预测 🎯

```
【大小预测】- 4个AI模型融合
模型1: 频率统计 (30%)
模型2: 趋势分析 (30%)
模型3: 周期分析 (25%)
模型4: RSI指标 (15%)

显示：
大  概率: 65.23%  置信度: 高
小  概率: 34.77%  置信度: 低

【单双预测】- 3个AI模型融合
显示概率和置信度

【波色预测】- 3个AI模型融合
显示三种波色的概率排序
```

### 功能3: 完整回测 🔄

```
支持4种回测类型：
1. ✅ 特码预测回测
2. ✅ 大小预测回测（新增）
3. ✅ 单双预测回测（新增）
4. ✅ 波色预测回测（新增）

操作：
1. 进入「历史回测」标签页
2. 选择回测类型
3. 点击「开始回测」
4. 查看准确率和详细记录

输出：
- 总测试数: 50
- 命中次数: 27
- 准确率: 54.00%
- 详细每期记录表
```

### 功能4: 自定义预测数量 📊

```
位置：左侧边栏 → 预测参数

融合预测数量: 1-49可调

影响：
✓ 预测结果显示数量
✓ 下注助手分配号码

推荐设置：

- 测试用: 可设置到49个
```

```

可调参数：

✓ 和值等

统计分析：
```

## 💡 使用技巧

### 技巧1: 合理设置预测数量

```
目的不同，设置不同：

学习算法：
- 融合预测: 10个
- Transformer: 10个
- 便于理解结果

研究趋势：
- 融合预测: 20-30个
- Transformer: 15-20个
- 扩大观察范围

压力测试：
- 融合预测: 49个
- Transformer: 49个
- 测试系统极限
```

### 技巧2: 下注助手档位调整

```
默认权重：
1档(最高): 3.0
2档(中): 1.5
3档(低): 0.5
4档(保本): 0.0

可根据风险偏好调整：

保守型：
1档: 2.0
2档: 1.0
3档: 0.3
4档: 0.0

激进型：
1档: 5.0
2档: 2.5
3档: 1.0
4档: 0.0
```

### 技巧3: 回测策略选择

```
不同策略，不同意义：

TOP1: 
- 最严格
- 准确率最低
- 但如果命中，利润最高

TOP3:
- 较严格
- 准确率较低
- 平衡风险和收益

TOP5/TOP10:
- 较宽松
- 准确率较高
- 但实际投注成本增加

建议：用TOP3做研究参考
```

### 技巧4: 历史记录分析

```
查看不同期数，观察规律：

最近10期：
- 查看短期趋势
- 观察连续性

最近30期：
- 查看中期模式
- 分析周期性

最近50-100期：
- 查看长期分布
- 验证统计规律
```
