## 🐛 常见问题排查

### 问题1: ModuleNotFoundError

**错误**:
```
ModuleNotFoundError: No module named 'xxx'
```

**解决**:
```bash
# 重新安装依赖
pip install streamlit pandas numpy scipy plotly scikit-learn openpyxl

# 或单独安装缺失的包
pip install xxx
```

### 问题2: 文件不在同一目录

**错误**:
```
ModuleNotFoundError: No module named 'lottery_core'
```

**解决**:
- 确保所有7个.py文件在同一文件夹
- 使用`cd`命令进入正确目录
- 在该目录中运行streamlit命令

### 问题3: Python版本过低

**错误**:
```
SyntaxError: invalid syntax
```

**解决**:
```bash
# 升级Python到3.8+
# 或使用虚拟环境
python -m venv venv
source venv/bin/activate  # Mac/Linux
venv\Scripts\activate     # Windows
```

### 问题4: 端口被占用

**错误**:
```
Port 8501 is already in use
```

**解决**:
```bash
# 指定其他端口
streamlit run lottery_app_v4_complete.py --server.port 8502
```

### 问题5: 准确率达不到90%

**可能原因**:
- 数据量不足（<500期）
- 数据质量差
- 随机性因素

**解决**:
1. 增加数据量到500-1000期
2. 检查数据完整性
3. 多运行几次
4. 85-89%也很好了
