## 🔧 技术实现

### 代码结构

```python
# 历史记录标签页
with tabs[6]:
    # 自定义期数输入
    custom_history_periods = st.number_input(
        "输入查看期数",
        min_value=10,
        max_value=2000,
        value=history_periods,  # 从侧边栏获取默认值
        step=10
    )
    
    # 快速选择
    quick_select = st.selectbox(
        "快速选择",
        options=["自定义", "最近30期", ..., "最近365期", "全部数据"]
    )
    
    # 处理快速选择
    if quick_select == "全部数据":
        custom_history_periods = len(st.session_state.data)
    elif quick_select != "自定义":
        custom_history_periods = int(quick_select.replace("最近", "").replace("期", ""))
    
    # 数据检查
    if custom_history_periods > len(st.session_state.data):
        st.warning("数据不足...")
        custom_history_periods = len(st.session_state.data)
    
    # 显示历史记录（使用custom_history_periods）
    history_df = HistoryViewer.get_recent_history(
        st.session_state.data, 
        custom_history_periods  # ← 使用自定义期数
    )
    
    # 显示统计分析（使用相同的custom_history_periods）
    history_stats = HistoryViewer.analyze_history(
        st.session_state.data, 
        custom_history_periods  # ← 完全同步
    )
```

## 📞 常见问题

### Q1: 侧边栏和标签页的期数设置有什么区别？

A: 
- **侧边栏**: 全局默认值，影响所有标签页
- **标签页内**: 局部设置，仅影响当前标签页
- **优先级**: 标签页内设置 > 侧边栏设置

### Q2: 我设置了365期，但统计图表只显示100期？

A: 
- 号码频率图：使用完整365期 ✓
- 走势图：限制为100期（保持清晰）
- 原因：100期以上走势图会很拥挤

### Q3: "全部数据"选项有什么用？

A: 
- 自动使用全部可用数据
- 无需手动输入总期数
- 适合完整统计分析

### Q4: 为什么有些地方最大是500，有些是2000？

A: 
- 回测期数：500期已足够验证
- 历史记录：2000期支持超长期查看
- 设计合理，满足不同需求
