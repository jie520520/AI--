**优点**:

- ✅ 命令行友好
- ✅ 适合自动化脚本

### 方式3: Python脚本（高级定制）

```python
from lottery_core import DataProcessor, PredictionEngine

# 加载数据
df = DataProcessor.load_excel('数据文件.xlsm')
data = DataProcessor.parse_data(df)

# 运行预测
engine = PredictionEngine(data)
predictions = engine.run_prediction(top_k=10)

# 打印结果
engine.print_predictions()
```

**优点**:
- ✅ 完全可定制
- ✅ 适合集成到其他项目
- ✅ 可以修改算法


4. **运行预测**
   - 点击「🚀 运行AI预测」按钮
   - 等待30-60秒
   - 查看结果！

5. **查看分析**
   - **预测分析**: AI融合预测 + Transformer预测
   - **特征工程**: 8维特征详细分析
   - **模型评估**: 5个模型性能对比
   - **历史回测**: 准确率测试

## ❓ 常见问题快速解答

### Q: 启动Web界面后浏览器没有自动打开？

**A**: 手动访问 `http://localhost:8501`

### Q: 提示找不到模块？

**A**: 重新安装依赖
```bash
pip install -r requirements.txt
```

### Q: Excel文件上传失败？

**A**: 检查：
1. 文件格式是否为 .xlsx, .xlsm, .xls
2. 工作表名称是否为「六合彩数据」
3. 列名是否正确（期号、开奖时间、号码1-6、特码）

### Q: 预测准确率如何？

**A**: 
- 理论随机: TOP1 ≈ 2%, TOP3 ≈ 6%, TOP5 ≈ 10%
- 模型可能略高，但**这不代表实际预测能力**
- 彩票完全随机，**不可预测**

## 🔧 故障排除

### 问题：ImportError: No module named 'xxx'

```bash
# 解决方案：安装缺失的模块
pip install xxx
```

### 问题：权限错误

```bash
# 解决方案：使用 --user 参数
pip install --user -r requirements.txt
```

### 问题：端口被占用

```bash
# 解决方案：指定其他端口
streamlit run lottery_app.py --server.port 8502
```

### 问题：内存不足

```bash
# 解决方案：减少数据量或增加系统内存
# 或者只使用最近的数据
# 在代码中修改: data = data.iloc[-500:]  # 只用最近500条
```
