## 📖 功能详解

### 1. 下注助手使用流程

```
1. 运行AI预测获得推荐号码
   ↓
2. 进入「下注助手」标签页
   ↓
3. 设置总预算和赔率
   ↓
4. 系统自动按置信度分配到4个档位
   ↓
5. 点击「计算最优下注金额」
   ↓
6. 查看每个号码的下注金额和预期利润
   ↓
7. 点击「复制投注格式」一键复制
```

**档位说明**：
- **1档（最高）**：置信度「高」的号码，权重3.0
- **2档（中）**：置信度「中」的号码，权重1.5
- **3档（低）**：置信度「低」的号码，权重0.5
- **4档（保本）**：保本号码，权重0.0

**保本原理**：
```python
基础投注 = 总预算 / 赔率
剩余资金 = 总预算 - (号码数量 × 基础投注)
额外分配 = 剩余资金 × (档位权重 / 总权重分数)
最终投注 = 基础投注 + 额外分配
```

### 2. 辅助预测AI模型

#### 大小预测（4模型融合）
```python
模型1: 频率统计（权重30%）
模型2: 趋势分析（权重30%）
模型3: 周期性分析（权重25%）
模型4: RSI指标（权重15%）

融合概率 = Σ(模型概率 × 权重)
```

#### 单双预测（3模型融合）
```python
模型1: 频率统计（权重40%）
模型2: 周期性分析（权重35%）
模型3: 模式识别（权重25%）
```

#### 波色预测（3模型融合）
```python
模型1: 频率统计（权重40%）
模型2: 趋势调整（权重40%）
模型3: 遗漏分析（权重20%）
```

### 3. 历史回测说明

**回测流程**：
```
1. 选择回测类型（特码/大小/单双/波色）
   ↓
2. 设置回测期数（10-200）
   ↓
3. 系统滚动预测每一期
   ↓
4. 对比预测值和实际值
   ↓
5. 计算命中率和准确率
```

**理论随机准确率**：
- 大小：50%
- 单双：50%
- 波色：33.3%
- 特码 TOP1：2%
- 特码 TOP10：20.4%

**模型准确率**：可能略高于随机，但这**不代表实际预测能力**！

### 4. 自定义预测数量

在侧边栏设置：

用途：
